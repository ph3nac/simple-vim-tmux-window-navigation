simple vim tmux window navigation
==========
This plugin is a simplified version of [vim-tmux-navigator](https://github.com/christoomey/vim-tmux-navigator)

Usage
---
add the following to your ~/.tmux.conf file:


Keybind
---

| key    | vim    | tmux |
| ------ | ------ | ---- |
| ctrl+h | ctrl+W | :.-  |
| ctrl+l | ctrl+w | :.+  |
